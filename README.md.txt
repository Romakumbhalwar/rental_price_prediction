House Rent Prediction

This project predicts the monthly rental price of residential houses based on 
features like location, area, number of rooms, furnishing type, and more. 
The model is built in a Jupyter Notebook and deployed using Streamlit on Render.

Live App
-> [https://rental-price-frontend.onrender.com]

Project Overview
- Trained a regression model on a rental dataset
- Feature engineering and preprocessing
- Final prediction using Linear Regression / Random Forest (as used)
- Simple Streamlit interface for user input and predictions

Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Fast API
- Jupyter Notebook(anaconda)
